Binding di Angular

            Binding ke attribute title pake {{ brace('') }}
          
    
          
    
            Binding ke attribute title pake [ ]
          
    
          
    
            Binding ke attribute title pake bind-(nama attribute)

  • Dari HTML ke COMPONENT (Event / Trigger)

              
    
            Event pake ( )
          
    
          
    
            Event pake on-(nama eventnya)
