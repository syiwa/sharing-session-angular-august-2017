Tips menampilkan data di Angular

  • Menggunakan {{ brace("nama_variabel") }}

    nama_variabel dideklarasi di component.ts 
              
    
            nama_variabel = "aWawaW"
          
    
            nama_variabel = "wWeeWW"
          
    
          {{ nama_variabel }}

  • Menggunakan {{ brace("nama_function()") }}

    nama_function() dideklarasi di component.ts & dikasih return 
            {{ ambilWaktu() }}

  • Bisa buat hitung-hitungan

    {{brace("1 + 1")}} = 2 
            {{ 1 + 01239213 * 291390123 / 123132 - 231231 * 231123  }}

  • Bisa buat unicode html

    <span [innerHTML]="string_html"></span>
    string_html adalah html yang dijadiin variabel di component.ts

  • Diapa-apain gampang

    Bisa pake pipe {{ brace('nama_variabel | uppercase') }} dll
    Pipe bisa custom juga 
            {{ 'uPPercAse' | uppercase }}
        {{ 'lOwErcaSe' | lowercase }}
        {{ variabel_tanggal | date: 'dd/mm/yyyy' }}

  • Gunakan ? ketika memanggil variabel object yang berpotensi undefined

    {{ brace('variabel_undefined?.name') }} Tidak akan berpengaruh ketika variabel_undefined adalah undefined 
            {{ variabel_undefined?.name }}
